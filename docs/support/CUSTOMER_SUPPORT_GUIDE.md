## üí° Pro Tips for Support Team

### Communication Tips
1. **Use simple Hindi/English** - Avoid technical jargon
2. **Be patient** - Many users are not tech-savvy
3. **Use analogies** - Compare to familiar concepts
4. **Repeat if needed** - Don't assume they understood
5. **Offer callbacks** - For complex issues

### Technical Tips
1. **Check app version first** - Many issues are fixed in updates
2. **Clear cache often helps** - Standard troubleshooting step
3. **Screenshots save time** - Ask for them early
4. **Device matters** - Some issues are device-specific
5. **Internet is key** - Most sync issues are connectivity related

### Emotional Intelligence
1. **Acknowledge frustration** - "Samajh sakta/sakti hoon ki ye frustrating hai"
2. **Take ownership** - "Main personally ensure karunga ye solve ho"
3. **Follow up proactively** - Don't wait for user to ping again
4. **Celebrate success** - "Bahut achha, ab sab kaam kar raha hai!"

## üÜò Emergency Procedures

### App Down / Server Issues
1. Check status page
2. Acknowledge to all users
3. Provide ETA if available
4. Update every 30 minutes
5. Apologize and compensate if extended

### Data Loss Report
1. Immediate escalation to L3
2. Get exact timeline
3. Check backups
4. DO NOT promise recovery
5. Keep user informed

### Security Incident
1. Immediate escalation to security team
2. Do not discuss details with user
3. Advise password change
4. Document everything
5. Follow security protocol
